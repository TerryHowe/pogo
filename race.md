# 12-Week Training Plan: 32-Mile Mountain Bike Race

## Phase 1: Base Building (Weeks 1–4)

| Day | Workout |
|-----|---------|
| Mon | Rest or easy 30-min run |
| Tue | Bike 1–1.5 hrs, moderate effort |
| Wed | Run 30–45 min, easy pace |
| Thu | Bike 1–1.5 hrs with hill intervals (4–6 x 3-min climbs) |
| Fri | Rest |
| Sat | Long bike ride: 1.5–2.5 hrs, progressively longer each week |
| Sun | Easy run 30–40 min (recovery) |

**Weekly volume:** 4–6 hrs bike, 1.5–2 hrs run

## Phase 2: Build (Weeks 5–8)

| Day | Workout |
|-----|---------|
| Mon | Rest or easy 30-min run |
| Tue | Bike 1.5–2 hrs with tempo blocks (2 x 15-min at race effort) |
| Wed | Run 40–50 min with 4–6 hill repeats |
| Thu | Bike 1.5–2 hrs, technical trail skills + intervals |
| Fri | Rest |
| Sat | Long bike ride: 2.5–3.5 hrs on terrain similar to race |
| Sun | Easy run 35–45 min |

**Weekly volume:** 6–9 hrs bike, 2–2.5 hrs run

**Week 8 long ride:** Aim for 28–32 miles to hit race distance once.

## Phase 3: Peak & Sharpen (Weeks 9–11)

| Day | Workout |
|-----|---------|
| Mon | Rest |
| Tue | Bike 1.5 hrs with race-pace intervals (3 x 10-min) |
| Wed | Run 35–45 min, moderate |
| Thu | Bike 1.5–2 hrs, mix of climbing and technical sections |
| Fri | Rest |
| Sat | Long ride: week 9 = 3 hrs, week 10 = 3.5 hrs, week 11 = 2.5 hrs |
| Sun | Easy run 30 min |

**Weekly volume:** 6–8 hrs bike, 1.5–2 hrs run

## Phase 4: Taper (Week 12)

| Day | Workout |
|-----|---------|
| Mon | Rest |
| Tue | Bike 45 min with a few short openers (3 x 2-min hard) |
| Wed | Easy run 20–25 min |
| Thu | Easy bike 30–40 min, spin the legs |
| Fri | Rest |
| Sat | **Race Day** |
| Sun | Recovery |

**Volume drops ~50% from peak week.**

## Key Principles

- **Long ride progression:** Add roughly 15–20% distance/time per week, with a cutback every 4th week
- **Running purpose:** Builds aerobic base and strengthens legs (especially quads/calves) without the joint stress of more saddle time
- **Specificity matters:** Do long rides on terrain that mimics race conditions — elevation, singletrack, loose surfaces
- **Intensity split:** ~80% easy/moderate, ~20% hard intervals
- **Nutrition practice:** Use long rides to dial in race-day fueling — aim for 200–300 calories/hr on rides over 2 hours

## Race Day Tips

- Pre-ride the course or study the map for technical sections and aid station locations
- Start conservative — 32 miles is long enough that blowing up early is a real risk
- Have a hydration/fueling plan and stick to it before you feel hungry or thirsty
